Get Started in Seconds

A simple, four-step process to simulation.

1. Choose a Module

Select from Logic Gates, Arithmetic, Logic Designer, or the full ALU simulator.

2. Provide Inputs

Toggle switches for gates, enter numbers for arithmetic, or type a boolean expression.

3. Run Simulation

Click 'Simulate' or watch the circuit update instantly as you make changes.

4. Analyze Results

Read the output from real-time LED displays and understand the process with step-by-step execution logs.
